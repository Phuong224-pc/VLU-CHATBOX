⚙️ Installation & Setup
Clone the repository: git clone https://github.com/your-username/vlu-chatbot.git

Create a file named config.js in the root directory.

Copy the content from config.example.js and paste it into config.js.

Replace "PASTE_YOUR_GROQ_API_KEY_HERE" with your actual Groq API Key.

Open index.html and start chatting!
